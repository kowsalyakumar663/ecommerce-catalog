# LUXE — E-Commerce Product Catalog
### Thiranex Capstone Project | Full-Stack Deployment & Project Architecture

---

## 🚀 Features

- **Modular frontend architecture** — Separated HTML, CSS, JS, and data files
- **Client-side routing** — Category filtering with seamless navigation (no page reload)
- **Live search** — Instant product search as you type
- **Sort & Filter** — Sort by price, name; filter by category
- **Shopping cart** — Add/remove items, quantity controls, persistent via localStorage
- **Responsive design** — Mobile-first, works on all screen sizes
- **Optimized assets** — Lazy-loaded images, minified-ready CSS/JS
- **Animations** — Smooth hover effects, page-load staggered reveals

## 🛠 Tech Stack

- HTML5 (semantic markup)
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (ES6+)
- Google Fonts (Playfair Display + DM Sans)
- Unsplash (optimized product images via URL params)

## 📁 Project Structure

```
ecommerce-catalog/
├── index.html          # Main app entry point
├── css/
│   └── style.css       # All styles (variables, components, responsive)
├── js/
│   ├── products.js     # Product data (16 products across 4 categories)
│   └── app.js          # App logic (filtering, cart, search, routing)
└── README.md
```

## 🌐 Deployment (Netlify — From Phone)

1. Go to **netlify.com** → Sign up with GitHub
2. Drag & drop the `ecommerce-catalog` folder onto the Netlify dashboard
3. Your live URL is ready in ~30 seconds! ✅

## 🌐 Deployment (Vercel — From Phone)

1. Upload project to GitHub (github.com → New repo → Upload files)
2. Go to **vercel.com** → Import your GitHub repo
3. Click Deploy → Live URL in ~1 minute ✅
