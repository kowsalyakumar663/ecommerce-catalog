##  Task 5 Requirements Covered
-  Modular frontend architecture

## Features

- Search & filter by category
- Add to cart & quantity control
- Order summary with discount
- Fully responsive on all devices

##  Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts
- localStorage (cart persistence)
